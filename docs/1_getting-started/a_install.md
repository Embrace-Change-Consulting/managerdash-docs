## Install
1. Download **ManagerDesk for macOS** from INSERT_LINK.
2. Open the `.dmg` and drag to **Applications**.

## First run: create your first alias
1. Click **Add Alias**.
2. Fill in **Name**, **Initials**, and **GitHub Alias**. Select a color then **Save**.
